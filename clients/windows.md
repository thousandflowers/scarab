# Installazione su Windows

1. Copia il file `certs/output/scarab-ca.pem`.
2. Fai doppio clic sul file per avviare la procedura guidata Installazione certificato.
3. Clicca su **Installa certificato...**.
4. Scegli **Computer locale**.
5. Scegli **Colloca tutti i certificati nel seguente archivio** e naviga su **Autorità di certificazione radice attendibili**.
6. Completa l'installazione. Windows ora fiderà di Scarab.
